# ubuntu-scripts

## Descrição
Scripts para instalação de ambiente de desenvolvimento Ubuntu.

- git
- docker
- docker-compose
- node/npm
- vscode
- chrome
- utilitários ubuntu

## Pré-requisitos
- Ubuntu 18.04

## Utilização
```
$ chmod +X developer-web.sh
$ ./developer-web.sh
```
Preste atenção pois o script pode solicitar sua interação.

### Arquivos individuais
- Composer
  - PHP
  - composer
```
$ chmod +X composer-install.sh
$ ./composer-install.sh
```
- React native
  - node/npm
  - react-native-cli
  - OpenJDK 8
  - Android studio
```
$ chmod +X react-native-install.sh
$ ./react-native-install.sh
```
