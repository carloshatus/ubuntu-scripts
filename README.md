# ubuntu-scripts

## Descrição
Script para instalação de ambiente de desenvolvimento Ubuntu.

- git
- docker
- docker-compose
- node/npm
- vscode
- chrome
- utilitários ubuntu

## Pré-requisitos
- Ubuntu 18.04

## Utilização
```
$ chmod +X developer-web.sh
$ ./developer-web.sh
```
Preste atenção pois o script pode solicitar sua interação.
